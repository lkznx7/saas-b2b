# SaaS B2B - Plataforma empresarial

Plataforma SaaS B2B para gestão de clientes, processos e automação comercial.

## Problema que Resolve

Pequenas e médias empresas precisam de uma ferramenta para gerenciar clientes, automatizar processos de venda e acompanhar métricas importantes para o negócio. A plataforma centraliza essas funcionalidades em uma solução web acessível.

## Tipo

SaaS Dashboard B2B

## Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS v3
- **Language:** TypeScript
- **部署:** Vercel

## Status

Em desenvolvimento

```bash
npm run dev
```
